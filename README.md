# Track Your Input

**Track Your Input** is a browser extension for Chrome and Firefox designed to help language learners quantify their progress by tracking listening comprehension time through video content.

---


## Features
* **Automatic Tracking:** Seamlessly logs videos as you watch them online.
* **Daily Goals:** Set and monitor personal immersion targets.
* **Total Time Tracker:** Keep an eye on your cumulative progress.
* **Immersion Calendar:** Get a visual overview of your input history from the past.
* **Multi-Language Support:** Track progress across multiple target languages simultaneously.
* **Detailed History:** Access an in-depth log of exactly what you have been watching.
* **Manual Entry:** Manually log time for offline or untracked activities.
* **Privacy First:** All your data is stored locally on your device.
* **Data Portability:** Easily export and import your data whenever needed.
* **Free & Open Source:** Built for the language learning community.

---

## Supported Platforms
The extension can track:
* **YouTube:** Full support for the main platform.
* **Embedded Media:** Works with YouTube videos embedded on other websites.
* **HTML5 Video:** Compatible with various websites that use the `<video>` HTML tag.

## How to Use
1. **Install** the extension for your preferred browser via the links above.
2. **Open the Extension** and configure your target languages.
3. **Start Watching** on YouTube or other supported video sites; the extension will automatically track your progress.
4. **Monitor Goals** by clicking the extension icon to see your daily stats and history.

## How to Build (Development)
This project is built with **Vite** and **React**. To build the extension from source, follow these steps:

<img width="75px" src="https://github.com/Pedro-Murilo/icons-for-readme/blob/main/.github/react-icon.svg" alt="ReactJS Icon" />

<img width="75px" src="https://github.com/Pedro-Murilo/icons-for-readme/blob/main/.github/typescript-icon.svg" alt="Typescript Icon" />


1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yoreni/track-your-input.git
   cd track-your-input
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Build the project:**
   ```bash
   npm run build
   ```

The compiled extension will be located in the dist folder, which can then be loaded into your browser as an unpacked extension.
