Track your Input is an Extension that works on Chrome and Firefox that helps languages learners track how much listening comprehension they have received through videos.

If you find bugs or have feedback you can make a github issue.

 Features
 - Automaticly Track Videos that you watch online
 - Daily Goals
 - Total time tracker
 - A calendar to give an overview of the amount of input from the past 
 - Stores Your data localy
 - Export and Import your data
 - Manualy Track Time
 - An indepth history that tells you excatly what you have been watching
 - Track multiple languages at the same time
 - Free & Open Source

Can track
 - YouTube
 - Embedded YouTube videos on other websites
 - Some websites that use the `<video>` HTML tag
